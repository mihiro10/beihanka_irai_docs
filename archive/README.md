# 🌾 米飯課への製造依頼システム (Production Request System)
このリポジトリは、惣菜課（生産管理）と米飯課の間で行われる製造依頼とスケジュール調整を管理するAppSheetアプリの公式ドキュメントおよびコンテキストデータを含んでいます。

## 🎯 リポジトリの目的
このリポジトリは、将来の保守担当者やAIアシスタント（Antigravity等）が、システムの設計思想・制約事項・仕様を即座に把握し、安全に開発・更新作業を行えるように整備された「コンテキスト（脳）」として機能します。

## 👥 対象ユーザー
1. **米飯課（現場作業・製造担当）:** 主に **iPad等のタブレット** を使用してスケジュールを確認し、自身の予定をブロックしたり、確定作業を行います。
2. **生産管理（スケジュール作成担当）:** 主に **PC（パソコン）** を使用してダッシュボードから依頼スケジュールを作成し、全体の稼働状況を監視します。

## 📂 ドキュメント構成 (`docs/`)
本システムの詳細な利用方法や構造については、`docs`フォルダ内のファイルを参照してください。
- [**01_user_manual.md**](./docs/01_user_manual.md) : ユーザー向け利用マニュアル（各デバイスでの操作方法・フィルター活用・グラフ確認など）
- [**02_system_architecture.md**](./docs/02_system_architecture.md) : システム仕様・稼働設計（MermaidによるDB・状態遷移図）・API連携について
- [**03_maintenance_guide.md**](./docs/03_maintenance_guide.md) : アプリ保守・マスタデータ更新・AppSheet制約（Valid If等）の管理マニュアル

## 🔌 AppSheet APIの外部アクセスについて
本システムはAppSheet APIを有効化して構築されています。これにより、Pythonスクリプトや社内の別システムから直接データベースの読み込み（Read）や新規スケジュールへの書き込み（Write）を行うことが可能です。
APIアクセスのデモやテストスクリプト群は `tests_and_data/` フォルダ内に格納されています。
